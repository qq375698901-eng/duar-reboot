# Town UI Set 01

这是一套给“城镇全 UI 化”方向准备的首版审查素材，视觉基调偏：

- 铁狱修道院
- 冷灰石砌结构
- 火烛暖金点缀
- 少量青绿色符文辉光

素材清单：

- `town_background_hub.svg`
- `town_panel_large.svg`
- `town_panel_side.svg`
- `town_button_primary.svg`
- `town_button_secondary.svg`
- `town_tab_active.svg`
- `town_icon_frame.svg`
- `town_ui_preview_board.svg`

建议用途：

- `town_background_hub.svg`：主城主页、职业大厅、铁匠铺等背景母版
- `town_panel_large.svg`：主内容区、大弹窗、成长页
- `town_panel_side.svg`：侧栏、任务栏、装备比较栏
- `town_button_primary.svg`：主 CTA，如“进入地牢 / 确认 / 升级”
- `town_button_secondary.svg`：次级操作，如“返回 / 取消 / 预览”
- `town_tab_active.svg`：页签高亮态
- `town_icon_frame.svg`：功能入口、货币、装备位图标底框
- `town_ui_preview_board.svg`：整套风格总览

当前默认假设：

- 城镇是静态分层 UI 场景，不是 2D 可跑动主城
- 风格延续地牢与监工/Boss 的冷硬中世纪审美，不做明亮王都或温暖酒馆路线
- 这一版优先验证美术方向，不是最终切图和九宫格生产版
