# 装备设计提案 01

日期：2026-03-26

## 文档目的

本文件用于提交第一批 5 件新装备提案，供审查与筛选。

当前工程口径下，“装备”暂时等于“武器”。因此本提案中的 5 件装备全部按武器设计，优先兼容现有长剑模组已经验证过的操作框架：

- 地面轻攻击三段
- 地面重攻击两段
- 奔跑轻攻击 / 奔跑重攻击
- 空中轻攻击 / 空中重攻击
- 守御
- 守御反击轻 / 重
- `Z + C`
- `Z + X`
- `Z + X + C`

本提案当前只做策划定义，不涉及代码实现，不假定背包、掉落、词缀生成 UI、装备栏 UI 已完成。

## 这一批装备的设计目标

- 先扩武器战斗风格，而不是扩系统复杂度
- 尽量复用现有武器模组已经支持的能力：位移、投射物、抓取、增益、超级护甲、守御反击
- 每件武器都要有清楚的战斗身份，避免“只是长剑换皮”
- 第一批武器尽量覆盖不同玩家偏好：稳健、爆发、机动、压制、控场

## 五件装备总览

| 名称 | 类型 | 关键词 | 上手难度 | 开发优先级建议 |
| --- | --- | --- | --- | --- |
| 铁狱短刀 | 短刀 | 快、贴身、追打 | 中 | 高 |
| 狱卫长枪 | 长柄枪 | 距离、直刺、拒近 | 中 | 高 |
| 断头巨斧 | 重斧 | 破防、重击、霸体 | 低 | 中高 |
| 余烬战镰 | 战镰 | 拉扯、扫荡、空中压制 | 高 | 中 |
| 朝圣手弩 | 轻弩 | 中距离、投射、风筝 | 中高 | 中 |

## 1. 铁狱短刀

### 核心定位

一把强调贴身压迫、连续追打与反打速度的轻型武器。它不是靠单次重击解决战斗，而是靠高频出手、快速切入、短后摇和反击节奏压死对手。

### 战斗体验关键词

- 近身
- 快切
- 连压
- 低单段伤害
- 高追击容错

### 推荐基础属性

- 基础攻击：7
- 基础防御：15%
- 重量：0.8
- Tier：1

### 基础操作气质

- `C` 三段应比长剑更短更快，移动性更强，允许边压边蹭位置
- `X` 两段不是纯慢重击，而是“突进刺击 -> 反手开膛”的快节奏重派生
- 跑攻应是非常实用的切入手段，而不是单纯补充动作
- 空中攻击应强调斜下压与贴地追打，方便连回地面

### 技能提案

- `Z + C` 影踏短突
  - 向前短距离突进斩
  - 命中后给予短僵直，方便立刻接普通 `C`
  - 目标作用：短距离抢回合、补刀、追击后撤敌人
- `Z + X` 锁喉擒切
  - 前冲两小段，命中后抓取，接一次近身处决式终结
  - 这是短刀最适合复用现有抓取链路的技能
  - 目标作用：把短刀的“刺客感”做出来
- `Z + X + C` 猎杀兴奋
  - 短时间攻速提高、跑攻前摇缩短、守御反击窗口略偏进攻
  - 目标作用：让短刀进入爆发压制期

### 设计重点

- 它必须是第一把真正鼓励玩家“黏着敌人打”的武器
- 它和长剑的区别不应只是速度更快，而是主动抢身位、抢先手、抢收招
- 它适合做成第一批新武器里手感最灵活的一把

### 风险提醒

- 如果动作过快而判定又太厚，容易直接压过长剑
- 必须用更短距离与更低单段伤害来换取流畅度

## 2. 狱卫长枪

### 核心定位

一把控制距离、依靠直线威胁和先手压制取胜的中距离武器。它要让玩家明显体验到“我在敌人进入我危险距离前先把他拦下来”。

### 战斗体验关键词

- 中距离
- 直刺
- 拒近
- 节奏严谨
- 起手优势

### 推荐基础属性

- 基础攻击：9
- 基础防御：20%
- 重量：1.4
- Tier：1

### 基础操作气质

- `C` 三段应以连续刺击为主，水平覆盖窄，但判定前伸明显
- `X` 两段应更像蓄势重刺和挑飞终结，强调读招后的一击穿线
- 跑攻应是一记低姿前冲突刺，适合先手截停
- 空中攻击不走大幅挥砍，而是向下或向前的点刺压制

### 技能提案

- `Z + C` 贯穿枪芒
  - 向前发射窄而长的线性枪气
  - 飞行速度快，宽度窄，突出“精准先手”
- `Z + X` 穿阵三连
  - 向前连续两段小突刺，最后一段重刺挑飞
  - 可复用现有 dash 连段结构，但不做抓取，改成纯穿刺连击
- `Z + X + C` 枪架稳持
  - 短时间强化直刺距离与守御稳定性
  - 可理解为进入更适合中距离卡点的架枪状态

### 设计重点

- 这把武器的乐趣来自“距离管理”，而不是堆动作花样
- 它应该明显比长剑更会打正面，更不擅长被贴脸后的混战
- 它很适合作为第二把优先开发武器，因为与长剑差异大，但实现边界清楚

### 风险提醒

- 如果给太宽的横向判定，会丢失长枪的识别度
- 必须允许它有明显盲区，不然会成为“比长剑更安全的长剑”

## 3. 断头巨斧

### 核心定位

一把围绕高伤害、强打断、厚重前摇和压迫感设计的重武器。它的吸引力来自“每次出手都很值”，而不是稳定连段。

### 战斗体验关键词

- 慢
- 重
- 霸体
- 大开大合
- 单次收益高

### 推荐基础属性

- 基础攻击：14
- 基础防御：35%
- 重量：2.8
- Tier：1

### 基础操作气质

- `C` 三段不是传统轻快连斩，而应是三段厚重横劈，前两段清空间，第三段重砸收尾
- `X` 两段应分别承担“强破防下砸”和“高击飞处刑”
- 跑攻应是非常明确的撞入式斜劈，承担开团功能
- 空中重击要有极强下坠压制感，最好是最凶的一类落地压制武器

### 技能提案

- `Z + C` 裂地重劈
  - 原地或微前移重砸，附带更强击飞 / 倒地倾向
  - 目标作用：打出明确的大收益窗口
- `Z + X` 破阵猛进
  - 带超级护甲向前顶进，接一次大范围处决斩
  - 最适合用来复用现有超级护甲开关能力
- `Z + X + C` 狂战姿态
  - 短时间提高伤害减免与攻击韧性，但牺牲机动性
  - 目标作用：让玩家进入“硬换血也赚”的状态

### 设计重点

- 这把武器应该让人感觉“我不是在连段，我是在砸碎对面”
- 它可以成为第一批里最容易读懂的一把重型武器
- 它和长剑的区别在于容错来源不同：长剑靠均衡，巨斧靠高收益与硬压

### 风险提醒

- 如果前后摇太长而缺少补偿，会玩起来笨重但不爽
- 必须给它明确的霸体、击飞和单次回报，不然只是“更慢的长剑”

## 4. 余烬战镰

### 核心定位

一把强调弧线覆盖、节奏扰乱和拉扯感的技巧型武器。它不是最稳的，也不是最直观的，但应该是最有“异质感”的一把。

### 战斗体验关键词

- 弧线
- 扫荡
- 拉扯
- 节奏错位
- 技巧型

### 推荐基础属性

- 基础攻击：10
- 基础防御：18%
- 重量：1.7
- Tier：1

### 基础操作气质

- `C` 三段应围绕大弧度斜扫、反向回钩、外圈收割展开
- `X` 两段应突出“把敌人从安全位置钩出来”与“挑空斩落”
- 跑攻应是贴地横掠，适合扫中多个目标或压边缘站位
- 空中攻击应该非常有特色，强调斜下收割和滞空控位

### 技能提案

- `Z + C` 回旋刃波
  - 发出短程缓速回旋波，覆盖面大于长剑剑气
  - 目标作用：制造控场与逼位
- `Z + X` 钩魂处刑
  - 前段把目标钩住或锁位，后段接一记近身收割
  - 如果当前抓取逻辑不适合做“拉回目标”，也可先做成命中后原地处决式二段
- `Z + X + C` 烬环加速
  - 短时间提高横扫速度与空中攻击存在感
  - 目标作用：把战镰的节奏优势放大

### 设计重点

- 这是第一批里最适合担当“高上限技巧武器”的一把
- 它的判定形状和挥动轨迹必须和长剑明显不同
- 它适合在动作和判定都成熟后再上，不建议作为第一把新原型

### 风险提醒

- 如果只是换成大一点的横扫，它会立刻退化成“长剑变体”
- 战镰的难点在于判定阅读和出手节奏，不能只靠特效表达差异

## 5. 朝圣手弩

### 核心定位

一把介于近战武器与远程武器之间的轻型器械。它不应该变成纯射击玩法，而应该是“用中距离射击辅助站位，再用短兵自保”的混合武器。

### 战斗体验关键词

- 中距离
- 点射
- 牵制
- 风筝
- 近远切换

### 推荐基础属性

- 基础攻击：8
- 基础防御：10%
- 重量：1.1
- Tier：1

### 基础操作气质

- `C` 三段应是近身弩体敲击、短刺、撤步射击这类混合动作
- `X` 两段应是更重的蓄力射击或穿透射击
- 跑攻应是滑步射击，突出边拉边打
- 空中攻击可设计为斜下急射或落地补射

### 技能提案

- `Z + C` 圣钉射
  - 发射飞行更快、击退更强的圣钉
  - 目标作用：作为稳定牵制手段
- `Z + X` 连弩突进
  - 先短距离滑步或翻身位移，再近距离连射 / 弩刃斩击
  - 目标作用：让轻弩不是纯站桩武器
- `Z + X + C` 装填过热
  - 短时间提高射击频率，但每次动作锁更严，形成高风险输出窗口
  - 目标作用：强化轻弩的节奏压制感

### 设计重点

- 轻弩必须仍然服从你当前横版动作战斗的近身框架
- 它应该是“有远程威胁的动作武器”，不是“切成另一个射击游戏”
- 这把武器适合在投射物体验已经稳定后再做

### 风险提醒

- 如果远程收益过高，会直接绕过近战读招系统
- 它必须保留近战弱点与装填节奏，不然会破坏当前战斗核心

## 第一批实现优先级建议

如果目标是尽快做出 5 把彼此差异明显、同时又不容易把项目拖进大改，那么推荐顺序如下：

1. 铁狱短刀
2. 狱卫长枪
3. 断头巨斧
4. 余烬战镰
5. 朝圣手弩

理由如下：

- 短刀最容易做出“和长剑完全不一样”的贴身压迫感
- 长枪最容易做出“距离管理”的差异
- 巨斧最容易做出“高收益重武器”的爽点
- 战镰很有特色，但对动作与判定表达要求更高
- 轻弩最有新鲜感，但也最容易碰到远程平衡问题

## 从当前工程角度的落地建议

下面这 5 件里，最适合先进入原型的有 3 件：

- 铁狱短刀
- 狱卫长枪
- 断头巨斧

原因：

- 这三件都可以主要依赖现有武器模组能力来实现
- 不强依赖新 UI、新资源系统、新状态异常系统
- 即便只先做白盒动作和基础命中效果，也能很快测出战斗定位

相对更建议后置的有 2 件：

- 余烬战镰
- 朝圣手弩

原因：

- 战镰更依赖特殊挥砍轨迹和判定阅读
- 轻弩更容易触碰远程节奏与投射物平衡边界

## 我当前的个人推荐

如果你希望第一批就同时覆盖“快、中、重”三种主流偏好，我最推荐先拍板这三件：

- 铁狱短刀
- 狱卫长枪
- 断头巨斧

然后把余烬战镰作为“技巧武器位”，朝圣手弩作为“中距离异类武器位”，留作第二轮审查。

## 待你审查的关键问题

你审这份提案时，可以优先看下面几件事：

- 这 5 件的战斗身份是否拉得足够开
- 有没有哪件在你心里不符合项目气质
- 你更想先做“快武器 / 长柄武器 / 重武器 / 技巧武器 / 远近混合武器”中的哪一种
- 你是否希望其中某一件更偏写实，或者更偏夸张
- 你是否希望第一批 5 件全部都是中世纪冷兵器，不要器械类

如果你审完愿意继续推进，下一步我建议直接把你拍板通过的 1 到 3 件，继续细化成“可实现规格文档”，拆到动作语义、数值区间、技能文本、原型优先级四个层面。
