# 黑杆银菱长枪设计文档

日期：2026-03-31  
状态：概念设计，未进入正式实现

## 文档定位
- 本文档用于定义新武器“黑杆银菱长枪”的战斗定位与实现边界。
- 设计参考对象是当前已实装长剑的装备思路与输入框架，不参考其他废案武器的造型或手感。
- 本文档先处理武器身份、招式方向、风险边界与接入建议，不处理掉落、词缀、强化、正式数值平衡。

## 设计目标
- 复用现有长剑已经验证过的输入结构，降低第二把可玩武器的接入成本。
- 明确把这把武器做成“距离管理与先手压制”型武器，而不是“更长、更安全的长剑”。
- 让玩家在使用时明显感受到攻击起点更远、直线压制更强、贴身处更难受。
- 保持中世纪地牢、低魔、实战化的气质，不做花哨枪缨和高幻想装饰。

## 外观结论
- 武器名称暂定：黑杆银菱长枪。
- 核心视觉特征：
  - 黑色长枪杆
  - 银色菱形枪头
  - 整体长度约为当前长剑贴图高度的两倍
- 气质关键词：
  - 冷硬
  - 纪律性
  - 守卫制式
  - 低魔实战兵器

## 与长剑的关系
- 长剑当前承担的是“稳健近战、均衡应对、正面连续压制”的基础武器位。
- 长枪不应继承长剑的横向容错与贴身稳定性。
- 长枪的价值应来自：
  - 更远的先手距离
  - 更明确的线性压制
  - 更强的站位管理回报
- 长枪的代价应明确表现为：
  - 贴身区更弱
  - 横扫覆盖更窄
  - 收招与转身容错更差

## 战斗身份
- 类型：长枪
- 定位：中距离先手控制武器
- 核心体验：在敌人进入长剑距离之前，用直刺、拦截和退枪节奏掌控交战距离。
- 上手门槛：中
- 主要奖励：
  - 先出手收益高
  - 对单体压制稳定
  - 能更早截停直线逼近型敌人
- 主要惩罚：
  - 被贴脸后不舒服
  - 横向清杂能力弱于长剑
  - 一旦贪刺落空，容易吃反打

## 输入框架
- 沿用当前长剑已验证的统一武器输入结构：
  - 地面轻攻击三段
  - 地面重攻击两段
  - 奔跑轻攻击
  - 奔跑重攻击
  - 空中攻击
  - 守御 / 守御反击
  - `Z + C`
  - `Z + X`
  - `Z + X + C`
- 结论：
  - 输入结构复用长剑
  - 命中意图、位移方向、判定形状和节奏全部重新定义

## 核心手感原则
- 优先“刺”而不是“挥”。
- 主伤害区集中在枪头前端，不给整条枪杆都能打人的宽泛判定。
- 多数招式允许轻微前探，但不应像长剑一样靠近身横斩维持稳定收益。
- 长枪的连段应强调“把对手留在枪尖附近”，而不是“贴脸压住不让动”。

## 基础招式设计

### 地面轻攻击三段
1. 第一段：试探前刺
- 用途：最稳定的站立起手。
- 表现：短前摇、直线小前探、枪头轻刺。
- 判定：细长矩形，起点略靠前，不覆盖脚边。
- 命中意图：僵直确认、抢先手。

2. 第二段：追压补刺
- 用途：在第一段命中或逼退后继续维持距离。
- 表现：收枪后再次向前送出，前探略大于第一段。
- 判定：仍然偏窄，但有效距离更长。
- 命中意图：延续压制，不追求高爆发。

3. 第三段：定步重刺 / 挑顶
- 用途：作为轻连段终结技，建立明显收益。
- 表现：可做成前刺后上挑，或直接强力贯刺。
- 建议方向：优先“重刺后微挑”，让它既像长枪终结，也保留一点对受击状态的推进价值。
- 命中意图：小击退、轻挑飞或较长僵直。

### 地面重攻击两段
1. 第一段：沉肩蓄力刺
- 用途：中距离抢先手、破对拼节奏。
- 表现：前摇明显，出手后向前送出很长一段。
- 判定：非常线性，前端强，近身弱。
- 命中意图：高伤直刺，强调距离感。

2. 第二段：撤步回枪后反送
- 用途：作为重击终结，制造更强控制收益。
- 表现：先轻微收枪，再以更重的力度贯出。
- 命中意图：明显击退或挑飞。
- 设计原因：
  - 让长枪重击与长剑重击区分开来
  - 突出“节奏控制”而不是单纯二连刺

### 奔跑轻攻击
- 方向：低位滑步刺
- 用途：跑动切入、抓后排、补刀。
- 重点：速度快、判定细、恢复不能太短。
- 风险：如果横向包容过大，会变成更好用的长剑跑轻。

### 奔跑重攻击
- 方向：长距离突进重刺
- 用途：高承诺先手。
- 重点：位移明显，命中收益高，空挥惩罚也要明显。
- 建议：只给线性突刺，不给大范围扫击。

### 空中攻击
- 建议主轴：空中下探刺
- 用途：从上往下压枪尖，处理高低差和落地抢点。
- 原则：
  - 不做大回旋
  - 不做华丽空转
  - 保持“枪尖先落地”的朴素危险感

## 守御与守御反击
- 守御能力应弱于长剑的稳定感。
- 理由：
  - 长枪的主价值在距离，不应同时拥有过稳的贴身防守。
- 建议表现：
  - 守御可用
  - 守御反击以短促顶刺为主
  - 反击距离有优势，但角度不宽

## 技能方向建议

### `Z + C`
- 方向：贯风刺 / 破列刺
- 表现：一次强化直刺，附带更远的线性命中。
- 用途：处理中距离突进、惩罚站桩敌人。
- 原则：不做弹道飞行物，保持“枪”而不是“法术枪”。

### `Z + X`
- 方向：连环逼退枪
- 表现：两到三次节奏明确的前送刺击，最后一击将敌人推出枪尖安全位。
- 用途：建立长枪专属的“距离重置”能力。
- 设计目标：让玩家打完后自然回到舒服间距，而不是继续贴脸。

### `Z + X + C`
- 方向：定锋架势
- 表现：短时进入强化架势，提高前端命中收益或对冲能力。
- 用途：用于房间内守线、站位压迫、对精英的正面试探。
- 原则：
  - 偏战术增益
  - 不做炫目爆发状态
  - 不把长枪做成纯数值怪

## 判定原则
- 有效命中区应主要集中在枪头与枪头后少量枪身范围。
- 玩家贴脸时，应存在明显“死角”或低收益区。
- 多数攻击框建议比长剑更长、更窄。
- 不建议给长枪大弧线宽判定，否则会直接变成更安全的长剑。

## 位移原则
- 长枪可以有小幅前探，强调压步。
- 但长枪不应像突进职业那样频繁穿人或贴身缠斗。
- 推荐方向：
  - 轻攻击：小前探
  - 重攻击：明显前探
  - 跑重：最大位移
- 所有位移都应服务于“把目标留在枪尖距离”，而不是钻进目标怀里。

## 原型数值建议
- 以下为首轮原型口径，只用于设计对齐，不代表最终平衡结论。

| 项目 | 长剑当前原型 | 黑杆银菱长枪建议 |
| --- | --- | --- |
| 基础攻击倾向 | 均衡 | 略高或持平 |
| 基础防御倾向 | 中 | 略低 |
| 攻击前摇 | 中 | 轻攻击略长，重攻击更长 |
| 攻击距离 | 中 | 明显更长 |
| 横向覆盖 | 中 | 更窄 |
| 贴身稳定性 | 中高 | 低 |
| 追击能力 | 中 | 中 |
| 先手能力 | 中 | 高 |

## 接入建议
- 物品名建议：黑杆银菱长枪
- 武器类型：`spear`
- 掌握路线 ID 建议：`spear`
- 图标建议：
  - 背包图标可直接使用当前贴图的放大预览版本
  - 若后续 UI 中识别度不足，再补专用图标裁切图
- 美术接入建议：
  - 当前贴图尺寸为 `16x96`
  - 若后续角色手持时遮挡或枢轴不合适，应先调挂点，不要先砍掉武器长度

## 关键风险
- 风险 1：判定太宽，变成“更安全的长剑”。
- 风险 2：贴身仍然强，导致长枪没有真实短板。
- 风险 3：前摇过长但收益不足，手感会变成“笨重但不值”。
- 风险 4：技能做成法术化，会偏离当前低魔实战化基调。

## 验收标准
- 玩家第一次上手时，能明确感受到“这不是长剑换皮”。
- 对单体正面逼近敌人时，长枪的截停价值清晰可见。
- 被近身后，玩家会自然意识到自己需要重新拉距离，而不是继续原地换血。
- 在清杂或横向压制上，长枪不应明显优于长剑。

## 当前结论
- 黑杆银菱长枪应作为“长剑之后的第二把可玩武器”优先候选之一。
- 它最重要的设计目标不是伤害更高，而是把“距离管理”正式引入玩家武器体验。
- 若进入实现阶段，第一版应先做最朴素、最清楚的直刺体系，确认手感成立后再补技能层变化。
